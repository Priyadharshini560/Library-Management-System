## Library Management System

## Description
This project is a console-based Library Management System developed using C++. It simulates basic library operations such as adding books, issuing and returning books, and searching records. The system uses object-oriented programming concepts for efficient management of library data.

## Features
Add new books
Display all books
Issue books
Return books
Search books by title or author
Menu-driven user interface

## Technologies Used
C++
Object-Oriented Programming (OOP)
Standard Template Library (STL - vector)

## How to Run

Compile the program:

g++ main.cpp -o library

Run the program:

./library
